This Library is primarily for the use of the Mendoza-Cortes Research group as well as any interested collaborators, priority for patches, assistance etc. will be given as such

Contents:
main
 - Methods for creating versatile/custom inputs for crystal22 in a scalable efficient manner

plotting
 - Methods for creating plotting input and output for band stuctures and density of states

heterostructures
 - Methods for creating Van der Waals heterostructures utilizing a modified version of supercell-core accounting for interlayer spacing and other parameters

workflow
 - Creation of dependency workflows for structures with several optimization steps, spin orbit coupling, plotting etc.
